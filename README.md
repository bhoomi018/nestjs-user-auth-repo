# NestJS User Auth API

A basic REST API built with **NestJS + MongoDB** that provides simple user authentication and profile management.  
Users can register, login with JWT, view and update their profile, upload an avatar, and soft delete their account.  
Admins can view all users with search & pagination.

**Features:**
- Register & Login (JWT)
- Get / Update Profile
- Soft Delete User
- Avatar Upload
- Admin: List All Users
- Clean folder structure with DTOs, Mongoose models, interceptors & filters

**Run Project:**
1. Install: `npm install`  
2. Create `.env`  
3. Start: `npm run start:dev`
